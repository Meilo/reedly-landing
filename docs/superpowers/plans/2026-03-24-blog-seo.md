# Blog SEO Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a blog with 5 SEO-targeted articles (FR + EN) to the Reedly Astro landing site.

**Architecture:** Astro 5 Content Collections with glob loader for Markdown files. Flat file structure (no lang subdirectories). Layout extended with optional props for lang, OG type, hreflang, and article metadata. Blog-specific CSS in a separate file.

**Tech Stack:** Astro 5, Markdown, CSS, vanilla JS (existing i18n system in `public/main.js`)

**Spec:** `docs/superpowers/specs/2026-03-24-blog-seo-design.md`

      "/pricing": "/tarifs",
      "/comparatif": "/comparison",
      "/comparison": "/comparatif",
    };
    var currentPath = window.location.pathname.replace(/\/$/, "");
    var targetPath = urlMap[currentPath];
    if (targetPath) {
      window.location.href = targetPath;
      return;
    }

    var mirrorMeta = document.querySelector('meta[name="blog-mirror"]');
